# 日常模拟-Skill

## 触发时机
当天是"日常日"（主要动线中无事件）时，Worker 需要编造角色一天的日常。

## 日常生成规则

### 基本原则
- 不要重复昨天完全一样的日程（吃了面今天就别吃面了）
- 日常要有生活气息，符合角色当前身份和阶段
- 角色的性格决定日常选择（福贵少爷 → 逛青楼赌钱；福贵农民 → 下地干活）

### 日常维度
按以下维度随机组合，保证多样性：

| 维度 | 示例变化 |
|------|---------|
| 健康 | 感冒、头痛、拉肚子、体检、睡懒觉 |
| 社交 | 朋友串门、邻居吵架、亲戚来访 |
| 兴趣 | 赌钱、喝酒、钓鱼、下棋、逛街 |
| 意外 | 丢东西、遇到熟人、被狗追、下雨淋湿 |
| 天气 | 下雨不出门、太热不想动、刮风吹倒东西 |
| 习惯 | 早起干活、中午打盹、晚上唠嗑 |

### 输出格式
每段日常总结为一句话概要，记录在今日日志（logs/YYYY-MM-DD.md）中。
例如："福贵今天睡到中午，去镇上赌了两把，输了五块大洋。"
